# Proposal Generator Prompt

## Template
```
Create a professional business proposal for {client} regarding {project}.

Include these sections:
1. Executive Summary (3-4 sentences highlighting key benefits)
2. Understanding of Client Needs (based on {pain_points})
3. Proposed Solution (how we'll address each pain point)
4. Timeline and Deliverables (clear milestones)
5. Investment and ROI (cost vs. value delivered)
6. Next Steps (simple call to action)

Tone: Professional, confident, client-focused
Length: 2-3 pages maximum
Budget: {budget}
Deadline: {deadline}
```

## Parameters
- **client**: Client company or individual name
- **project**: Project description or scope
- **pain_points**: 3-5 specific problems the client is facing
- **budget**: Project budget or price range
- **deadline**: Project completion date

## Example
```
Create a professional business proposal for Tech Startup Inc regarding website redesign.

Include these sections:
1. Executive Summary
2. Understanding of Client Needs (based on slow loading times, poor mobile experience, outdated design)
3. Proposed Solution
4. Timeline and Deliverables
5. Investment and ROI
6. Next Steps

Tone: Professional, confident, client-focused
Length: 2-3 pages maximum
Budget: $15,000
Deadline: 2026-05-15
```

## 🚀 Quick Start

### 1. Use the Template
Visit the [GitHub repository](https://github.com/shaguoerai/ai-business-writing-pipeline) and click **"Use this template"**.

### 2. Clone Your Repository
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 3. Generate Your First Content
```bash
# Install dependencies
pip install -r requirements.txt

# Generate a professional proposal
python scripts/generate-content.py \
  --type proposal \
  --client "Tech Startup Inc" \
  --project "Website Redesign" \
  --budget "$15,000"
```

### 4. Set Up Automation
Configure GitHub Actions to run on schedule (daily, weekly, monthly).
